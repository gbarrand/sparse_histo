go-hsparse
==========

`go-hsparse` is a simple test-bed for `github.com/go-hep/dao` (a data
analysis library.)

## installation

Install `go` and then:

```sh
$ go get github.com/gbarrand/sparse_histo/...
```

## Use

```sh
$ go-hsparse
 h1d.Fill(10000000) = 656.092226ms (mean=-0.500000000003953 rms=2.8722813232916278)
```
